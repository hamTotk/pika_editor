# pika インストールガイド

Windows 向け超軽量 Markdown/HTML エディタ。AIエージェントの出力物を確認・差分レビューする伴走ツール。

- **対応環境**: Windows 11 x64 のみ（UI は日本語）
- **WebView2 ランタイム**: Windows 11 には標準で同梱されています。万一不在の環境では、インストーラーが
  自動で取得します（ポータブル版・起動時に不在を検出した場合はアプリが導入を案内します）。
- **コード署名なし・自動更新なし**（後述のリスクを参照）

配布形態は2つあります。用途で選んでください。

| 形態 | 向いている用途 | 関連付け | データの場所 |
|---|---|---|---|
| **インストーラー版**（`pika_<version>_x64-setup.exe`） | 自分の PC に常用する | エクスプローラー統合あり | `%LOCALAPPDATA%\pika\` |
| **ポータブル版**（`pika-<version>-portable.zip`） | USB 等で持ち運ぶ・PC に痕跡を残したくない | 登録しない | 展開フォルダ内 `pika-data\` |

---

## インストーラー版

### 1. SmartScreen 警告の回避（重要）

pika は**コード署名をしていない**ため、初回起動時に Microsoft Defender SmartScreen が
「**WindowsによってPCが保護されました**」という青い画面を出すことがあります。これは未署名アプリ共通の
警告で、pika が危険という意味ではありません。次の手順で実行できます。

1. 警告画面の「**詳細情報**」をクリックする
2. 下部に現れる「**実行**」ボタンをクリックする

> セットアップ実行ファイルを右クリック →「プロパティ」→ 全般タブ下部の「**許可する（ブロックの解除）**」に
> チェック → OK、でも警告を抑止できます。

### 2. インストール

1. `pika_<version>_x64-setup.exe` を実行する
2. **インストール範囲を選ぶ**（インストーラーの最初のページ）:
   - **現在のユーザーのみ**（推奨・既定）… **管理者権限不要**。`%LOCALAPPDATA%\Programs\pika` に入る
   - **すべてのユーザー**（全ユーザー共通）… **管理者権限（UAC 昇格）が必要**。`C:\Program Files\pika` に入る
3. 画面の指示に従って進める
4. 完了するとスタートメニューに「pika」が登録されます

> **「すべてのユーザー」を選んだ場合の注意**: 本体は全ユーザーで共有されますが、**エクスプローラー統合
> （関連付け・右クリック）と設定/状態/スナップショットは「インストールした本人」だけ**に適用されます
> （登録先が現在のユーザー領域＝HKCU / `%LOCALAPPDATA%` のため）。他のユーザーで関連付けを使うには、
> そのユーザーで一度 `pika-cli.exe --register-shell` を実行してください。迷ったら「現在のユーザーのみ」で
> 問題ありません。

### 3. エクスプローラー統合を有効にする（任意）

インストール時に、関連付けの**候補**登録と右クリックメニューが自動で追加されます。

- **右クリック →「pikaで開く」**: ファイル・フォルダの右クリックメニューから直接開けます（すぐ使えます）
- **ダブルクリックで pika を既定にする**: Windows 11 は既定アプリをアプリ側から強制設定できない仕様
  （ユーザー選択がハッシュ保護されている）ため、**最後はご自身で一度だけ既定に設定**してください。

  1. `.md` / `.markdown` / `.html` / `.htm` のいずれかを右クリック
  2. 「**プログラムから開く**」→「**別のプログラムを選択**」
  3. 一覧から **pika** を選び、「**常にこのアプリを使って .md ファイルを開く**」にチェック → OK

## ポータブル版

1. `pika-<version>-portable.zip` を任意のフォルダ（USB メモリ等でも可）に展開する
2. フォルダ内の `pika.exe` を実行する（SmartScreen 警告が出たら上記と同じ手順で「実行」）

ポータブル版の特徴:

- フォルダ内に `portable.txt` がある間、設定・状態・スナップショット・ログは**展開フォルダ内の
  `pika-data\` にのみ保存**されます。実行した PC の `%LOCALAPPDATA%` には何も書き込みません。
- **エクスプローラー統合（関連付け・右クリック）は登録しません**（実行ホストに痕跡を残さない方針）。
- フォルダごと削除すれば完全に消えます（アンインストール作業は不要）。

---

## データの保存場所

pika は**ワークスペース（開いたフォルダ）の中に管理ファイルを一切作りません**。すべてデータルート配下に置きます。

| 種類 | インストーラー版 | ポータブル版 |
|---|---|---|
| 設定 `settings.toml` | `%LOCALAPPDATA%\pika\` | `<展開先>\pika-data\` |
| 状態復元 `state.json` | 同上 | 同上 |
| 退避スナップショット `snapshots\` | 同上 | 同上 |
| 診断ログ `logs\` | 同上 | 同上 |

- **設定**: `settings.toml` を直接編集します（GUI 設定画面は持ちません）。pika 側からは書き戻しません。
- **ログ**: 不具合報告に使えます。メニューからログフォルダを開けます（ファイル内容は記録しません・パスと操作のみ）。
- **スナップショット**: 外部変更や上書きから復元するためのバックアップです（最後の砦・既定上限 500MB）。

## アンインストール

### インストーラー版

「設定」→「アプリ」→「インストールされているアプリ」から **pika** をアンインストールします。

- エクスプローラー統合（関連付け・右クリック）は自動で**完全に解除**されます（残骸を残しません）。
- **設定・状態・ログ**は削除されます。
- **退避スナップショット**は、アンインストール時に「残す／消す」を尋ねられます（**既定は「残す」**）。
  再インストールやデータ救出の可能性に備え、明示的に消さない限り残ります。

### ポータブル版

展開したフォルダを削除するだけです（`pika-data\` も一緒に消えます）。

## 既知の制約・リスク（ご確認ください）

- **未署名配布**: コード署名をしていないため SmartScreen 警告が出ます（上記手順で回避）。署名証明書の
  コストと配布規模を踏まえた判断で、初期版は署名しません。配布元（正規の入手先）が信頼できることを
  確認のうえご利用ください。
- **自動更新なし**: 新版は手動で入れ替えてください。状態ファイルはバージョン非互換時に安全側（読まない・
  壊さない）に倒すため、新旧の混在で既存データが壊れることはありません。
- **同梱 JS/CSS（Mermaid/KaTeX/ハイライト）は更新されません**: 外部通信を遮断しているため、重大な脆弱性が
  出た場合はパッチ版の配布で対応します。
- **同梱 OSS ライセンス**: 本体に同梱する第三者 OSS のライセンス表示は `THIRD_PARTY_NOTICES`（配布物に同梱）
  および pika の「バージョン情報」から確認できます。

## 補足: コマンドライン（エージェント向け）

`pika <パス>` でフォルダ/ファイルを開けます（既に起動済みなら同じウィンドウへ転送）。`pika -g <行:桁> <ファイル>`
で位置指定、`pika --help` / `--version` も利用できます。詳細は要件定義書 3 章を参照してください。
